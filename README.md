## Project Description
This project expands the Part 1 Cybersecurity Awareness Chatbot into a WPF graphical user interface. The chatbot now uses a GUI, voice greeting, ASCII art, keyword recognition, random responses, memory, sentiment detection, conversation flow, and error handling.

## Features Implemented
- WPF graphical user interface
- Voice greeting using `SoundPlayer`
- ASCII art displayed in the GUI
- Keyword recognition for password, phishing, privacy, scam, malware, and 2FA
- Random responses using dictionaries and lists
- Conversation flow for follow-up questions like "tell me more", "why", and "give me another tip"
- Memory feature for the user's name and favourite cybersecurity topic
- Sentiment detection for worried, confused, positive, and neutral messages
- Default error response for unknown input
- Object-oriented structure using models and services

## How to Run
1. Open the project folder in Visual Studio 2022.
2. Make sure the target framework is supported: .NET 8 Windows.
3. Confirm that `Greeting.wav` is inside the `Media` folder.
4. Build the solution.
5. Run the project.
6. Start by typing: `My name is Sizolwethu`.
7. Ask questions such as:
   - Tell me about password safety
   - Give me a phishing tip
   - I am interested in privacy
   - Tell me more
   - Why?
   - I am worried about scams
